## 1. 登录认证模块

```mermaid
flowchart LR
    subgraph 前端["前端 LoginPage"]
        A[输入账号 + 密码]
        B[POST /api/auth/login]
        C{登录成功?}
        D[存储 token, 跳转主页]
        E[显示错误提示]
    end

    subgraph 后端["后端 auth.py"]
        F[查询用户]
        G{用户存在且未禁用?}
        H{密码正确?}
        I[密码升级: 明文→SHA256]
        J[签发 JWT<br/>120分钟过期]
        K[返回 token + 用户信息]
    end

    A --> B --> F --> G
    G -->|否| E
    G -->|是| H -->|否| E
    H -->|是| I --> J --> K
    C -->|200| D
    C -->|非200| E
    K --> C

    style D fill:#d4edda
    style E fill:#f8d7da
```

```mermaid
flowchart LR
    subgraph 鉴权["每次请求 @jwt_required"]
        R[请求带 Bearer Token] --> P[解析 JWT]
        P --> V{Token 有效<br/>未过期?}
        V -->|否| X[返回 401/403]
        V -->|是| U{用户存在<br/>status=1?}
        U -->|否| X
        U -->|是| L{拥有所需角色?}
        L -->|否| X
        L -->|是| G[设置 g.current_user<br/>请求通过]
    end

    style G fill:#d4edda
    style X fill:#f8d7da
```

## 3. 单据解析模块

```mermaid
flowchart TB
    A[上传 PDF] --> B[INSERT pdf_parse_task]
    B --> C[ZhipuAI 上传文件<br/>创建解析任务]
    C --> D[轮询 task_id<br/>等待结果]
    D --> E{succeeded?}
    E -->|否| F{超时?}
    F -->|否| D
    F -->|是| G[解析失败]
    E -->|是| H[获取 content 文本]

    H --> I[normalize_text<br/>OCR 纠错]
    I --> J[parse_header_fields<br/>解析头部字段]
    J --> K[parse_fee_items<br/>解析费用明细]
    K --> L[upsert customs_business<br/>按 s_ref 新增或更新]
    L --> M[INSERT customs_business_fee_item]
    M --> N[更新状态: SUCCESS]

    G --> P[更新状态: FAILED]
    N --> R[返回给前端]
    P --> R

    style N fill:#d4edda
    style P fill:#f8d7da
    style R fill:#d4edda
```

## 5. 费用核算模块

```mermaid
flowchart TB
    subgraph 前端["前端 CostAnalysisView"]
        I[录入: 海关费/退税/美元额<br/>汇率/其他费/商品数量]
        I --> C[POST /api/cost/calculate]
    end

    subgraph 后端["后端 cost.py"]
        B1["净海关费 = 海关费 - 退税"]
        B2["美元折算 = 美元额 × 汇率"]
        B3["总成本 = 净海关费 + 美元折算 + 其他费"]
        B4["单件成本 = 总成本 ÷ 总数量"]
        B5[按数量比例分摊到各商品]
        B6[返回计算结果]
    end

    C --> B1 --> B2 --> B3 --> B4 --> B5 --> B6

    subgraph 汇率["汇率获取"]
        R1[GET 实时汇率 API]
        R1 --> R2{成功?}
        R2 -->|是| R3[upsert fx_rate_cache]
        R2 -->|否| R4[读缓存]
        R4 -->|无缓存| R5[返回默认值 5.0]
        R3 & R4 --> R6[返回汇率]
    end

    style R6 fill:#d4edda
    style B6 fill:#d4edda
```

```mermaid

    subgraph 状态推导["整体状态推导"]

        U2 --> U3[重新推导整体状态]

    end

        V --> E[点击编辑]
        E --> S[选择状态 + 填日期备注]
        S --> P[PUT 保存]
    end

    style CLEARED fill:#d4edda
    style INSP fill:#fff3cd
```

## 7. 任务管理模块

```mermaid
flowchart TB
    T1[触发解析任务] --> T2[INSERT pdf_parse_task<br/>status=PROCESSING]
    T2 --> T3[轮询智谱 AI]
    T3 --> T4{succeeded?}
    T4 -->|是| T5[保存 content<br/>status=SUCCESS]
    T4 -->|否| T6{failed?}
    T6 -->|是| T7[保存 error<br/>status=FAILED]
    T6 -->|否| T8[超时或继续等待]
    T8 --> T3

    T5 & T7 --> T9[更新 pdf_file 状态]

    T9 --> Q[GET /api/tasks/:id<br/>查询任务详情]
    Q --> Q2[关联查询<br/>文件 + 业务记录]
    Q2 --> Q3[返回完整信息]
```
