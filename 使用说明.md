# PDF数据提取与数据库存储系统

## 📋 功能说明

本系统可以从PDF文件中提取结构化数据，并以格式化的方式展示在屏幕上，同时支持将数据存入数据库。

## 🚀 快速开始

### 1. 提取并显示PDF数据

```bash
python3 pdf_to_json.py
```

**输出内容：**
- 🏢 公司信息（名称、地址、CNPJ）
- 📋 单据信息（编号、开具日期、到期日期）
- 👤 客户信息（名称、CNPJ、地址等）
- 📦 流程信息（流程号、出口商、发票号、金额等）
- 💰 费用明细表（日期、代码、描述、金额、类型）
- 💵 总计信息（贷方、借方、余额）

### 2. 提取数据并存入数据库

```bash
python3 save_to_database.py
```

**功能：**
- 自动创建SQLite数据库
- 创建5个数据表（documents, companies, clients, processes, expense_items）
- 插入提取的数据
- 查询验证数据
- 显示统计信息

## 📊 数据库结构

### documents表 - 文档主表
```sql
- document_number (TEXT) - 单据编号
- issue_date (TEXT) - 开具日期
- due_date (TEXT) - 到期日期
- total_credit (REAL) - 贷方总计
- total_debit (REAL) - 借方总计
- balance (REAL) - 余额
```

### companies表 - 公司信息
```sql
- name (TEXT) - 公司名称
- address (TEXT) - 公司地址
- cnpj (TEXT) - 公司CNPJ
```

### clients表 - 客户信息
```sql
- name (TEXT) - 客户名称
- cnpj (TEXT) - 客户CNPJ
- address (TEXT) - 地址
- city (TEXT) - 城市
- district (TEXT) - 区域
- state (TEXT) - 州
- cep (TEXT) - 邮编
```

### processes表 - 流程信息
```sql
- document_number (TEXT) - 关联单据号
- process_number (TEXT) - 流程编号
- exporter (TEXT) - 出口商
- mawb (TEXT) - 主运单号
- di_number (TEXT) - DI编号
- invoice_number (TEXT) - 发票号
- cif_brl (REAL) - CIF金额(巴西雷亚尔)
- fob_usd (REAL) - FOB金额(美元)
- cif_usd (REAL) - CIF金额(美元)
- exchange_rate_usd (REAL) - 美元汇率
```

### expense_items表 - 费用明细
```sql
- document_number (TEXT) - 关联单据号
- expense_date (TEXT) - 费用日期
- expense_code (TEXT) - 费用代码
- description (TEXT) - 描述
- amount (REAL) - 金额
- transaction_type (TEXT) - 类型(credit/debit)
```

## 🔧 代码示例

### 仅提取数据（不存数据库）

```python
from pdf_to_json import PDFDataExtractor

# 创建提取器
extractor = PDFDataExtractor("P1.pdf")

# 提取所有数据
data = extractor.extract_all_data()

# 格式化打印
extractor.print_formatted_data(data)

# 获取数据库格式的记录
db_records = extractor.get_database_records(data)

# 访问特定数据
print(f"单据编号: {data['header']['document']['number']}")
print(f"公司名称: {data['header']['company']['name']}")
print(f"费用项目数: {len(data['items'])}")
```

### 自定义数据库操作

```python
from save_to_database import DatabaseManager
from pdf_to_json import PDFDataExtractor

# 提取数据
extractor = PDFDataExtractor("P1.pdf")
data = extractor.extract_all_data()
db_records = extractor.get_database_records(data)

# 连接数据库
db = DatabaseManager("my_database.db")
db.connect()
db.create_tables()

# 插入数据
db.insert_data(db_records)

# 查询文档
db.query_document("12626")

# 统计信息
db.get_statistics()

# 关闭连接
db.disconnect()
```

### 使用其他数据库（MySQL示例）

```python
import mysql.connector

# 连接MySQL
conn = mysql.connector.connect(
    host="localhost",
    user="your_user",
    password="your_password",
    database="pdf_data"
)
cursor = conn.cursor()

# 插入数据
for doc in db_records['documents']:
    cursor.execute("""
        INSERT INTO documents (document_number, issue_date, due_date, total_credit, total_debit)
        VALUES (%s, %s, %s, %s, %s)
    """, (doc['document_number'], doc['issue_date'], doc['due_date'], 
          doc['total_credit'], doc['total_debit']))

conn.commit()
conn.close()
```

## 💡 提示

1. **数据格式**: 程序会自动清理PDF特殊字符（如 cid:0）
2. **日期格式**: 保持原始格式（DD.MM.YYYY 或 DD/MM/YYYY）
3. **金额格式**: 自动转换为浮点数，便于计算
4. **唯一性**: CNPJ和单据编号使用UNIQUE约束，避免重复
5. **扩展性**: 可轻松切换到MySQL、PostgreSQL等数据库

## 🐛 常见问题

**Q: 提取的某些字段为空？**
- 检查PDF格式是否与正则表达式匹配
- 在 `extract_header_info()` 中调整模式

**Q: 如何批量处理多个PDF？**
```python
import glob

for pdf_file in glob.glob("*.pdf"):
    extractor = PDFDataExtractor(pdf_file)
    data = extractor.extract_all_data()
    # 处理数据...
```

**Q: 如何导出为Excel或CSV？**
```python
import pandas as pd

# 将费用明细导出为Excel
df = pd.DataFrame(data['items'])
df.to_excel('expenses.xlsx', index=False)
```
